# Royal Chess

FIDE準拠のチェスアプリ。Stauntonスタイルの駒、PWA対応でオフラインプレイ可能。

## Features

- FIDE Standard Rules (2023)
- Staunton Style SVG Pieces
- 2 Player / vs Computer (AI)
- PWA - Offline Support
- Mobile Responsive
- Drag & Drop / Touch Support

## Play

Open `chess.html` in your browser, or host on any static server.

### GitHub Pages

1. Fork this repository
2. Go to Settings > Pages
3. Select "main" branch and save
4. Access at `https://[username].github.io/[repo-name]/chess.html`

## Special Rules Supported

- Castling (King-side & Queen-side)
- En Passant
- Pawn Promotion
- Check / Checkmate / Stalemate Detection
